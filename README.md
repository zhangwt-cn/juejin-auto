# juejin-auto
> 稀土掘金自动化任务，例如签到等功能。

# features

- [x] 掘金自动化签到
- [x] 钉钉消息推送
- [x] Server酱微信公众号消息推送
- [x] Telegram 消息推送

# use
1. 首先需要将项目 `fork` 到自己的 `github` 仓库中
2. 进入项目主页点击 `Settings` > `Security` > `Secrets` > `Actions`  配置信息, [查看更多](https://github.com/top-zhang/juejin-auto/wiki/配置)
3. 定时任务配置了每天9点运行，但是由于 github actions 并不准时，大概在12点左右才能执行
